## 3. ПРОБЛЕМЫ ТЕКУЩИХ ТЕСТОВ

### Критические ошибки

1. **Async fixture bug** в `test_e2e_playwright.py`
   ```python
   # НЕПРАВИЛЬНО:
   @pytest.fixture
   async def page(context: BrowserContext) -> Page:
       page = await context.new_page()
       yield page
       await page.close()
   
   # Ошибка: AttributeError: 'async_generator' object has no attribute 'goto'
   ```

2. **Отсутствуют тесты для новых функций:**
   - Группировка нарушений (`/grouped` endpoint)
   - Оптимизированная таблица (90% экономия)
   - Цветовая индикация частоты
   - Пагинация на frontend

3. **Нет тестов на responsive:**
   - Mobile viewport (360x640)
   - Tablet viewport (768x1024)
   - Desktop viewport (1920x1080)
