# LinguaCheck-RU

Система автоматического контроля соблюдения требований к использованию русского языка в публичном пространстве (ФЗ №168-ФЗ).

## Основные возможности

1.  **Сканирование сайтов (Crawler)**:
    - Глубокий обход страниц сайта (BFS) с учетом домена.
    - **Языковая фильтрация (Phase 9)**: Автоматический пропуск нерусскоязычных страниц (на базе плотности кириллицы) для чистоты отчетов.
    - Извлечение текста и анализ CSS-свойств.
    - Автоматическая фиксация визуального доминирования иностранных слов.
    - **Визуальные доказательства (Phase 6)**: Автоматическое создание скриншотов конкретных нарушений.
2.  **Интеллектуальный анализ текста**:
    - **Фильтрация исключений**: Игнорирование имен собственных, городов и общепринятых сокращений (ул., обл. и т.д.).
    - **Потенциальные бренды (Phase 8)**: Иностранные слова с заглавной буквы помечаются предупреждением `possible_trademark` вместо ошибки, если они не зарегистрированы как ТМ.
    - Морфологический анализ через `pymorphy3` и сверка с нормативными словарями.
3.  **Управление результатами**:
    - **История проверок**: Сохранение и быстрый доступ к ранее проведенным сканированиям.
    - **Продвинутая фильтрация**: Поиск по слову, контексту или **URL страницы**, фильтрация по типам нарушений.
    - Экспорт отчетов в CSV и PDF.
4.  **Проверка текстов и файлов**:
    - Веб-интерфейс для вставки текста и загрузки PDF-файлов.
5.  **Словарная база**:
    - Использование нормативных словарей (Орфографический, Толковый, Иностранных слов).
    - База зарегистрированных товарных знаков (`trademarks`).

## Технологический стек

- **Backend**: FastAPI, SQLAlchemy (PostgreSQL/SQLite), Playwright (скрапинг), Pymorphy3.
- **Frontend**: React, Mantine UI, Tabler Icons.
- **Data**: Обработка PDF словарей через `pdfminer.six`.

## Установка и запуск (Локально)

### Бэкенд

1. Перейдите в папку `backend`.
2. Создайте окружение: `python -m venv .venv`.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Установите браузеры: `playwright install chromium`.
5. Запустите сервер: `python run.py`.

### Фронтенд

1. Перейдите в папку `frontend`.
2. Установите зависимости: `npm install`.
3. Запустите: `npm run dev`.

### Импорт словарей

Для наполнения базы используйте скрипт:
`python scripts/import_dictionaries.py "путь_к_pdf" --dictionary-name [Orthographic|ForeignWords|Explanatory|Orthoepic] --version "2024"`
