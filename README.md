# LinguaCheck-RU

Система автоматического контроля соблюдения требований к использованию русского языка в публичном пространстве (ФЗ №168-ФЗ).

## 📋 О проекте

LinguaCheck-RU автоматически проверяет сайты и тексты на соответствие требованиям Федерального закона №168-ФЗ «О государственном языке»:

- Приоритет русского языка в публичном пространстве
- Отсутствие иностранных слов без кириллического сопровождения
- Текстовый анализ нарушений

## ✨ Функциональные возможности

Система предоставляет полный цикл инструментов для контроля чистоты языка:

- **🔍 Сканирование сайтов**:
  - Автоматический обход страниц (глубина до 5 уровней).
  - Рендеринг JavaScript через Playwright для анализа динамического контента.
  - Умная фильтрация: игнорирование страниц с низким содержанием кириллицы.
- **📝 Анализ текста и файлов**:
  - Мгновенная проверка вставленного текста.
  - Поддержка загрузки документов: **TXT, DOCX, PDF**.
  - Морфологический разбор слов с использованием `pymorphy3`.
- **📚 Нормативные словари**:
  - Интеграция с официальными словарями (Орфографический, Толковый, Иностранных слов и др.).
  - Динамический расчет объема словарной базы.
- **🛡 Управление исключениями**:
  - **Глобальные исключения**: список слов, которые никогда не помечаются как нарушения.
  - **Товарные знаки**: ведение реестра брендов для исключения ложных срабатываний.
- **📊 Отчетность и UX**:
  - Группировка и сортировка нарушений по страницам и типам.
  - Экспорт детальных отчетов в **Excel (XLSX)** и **PDF**.
  - Современный интерфейс с темной темой, сворачиваемым сайдбаром и SEO-оптимизацией.

## 🚀 Запуск локально

### Предварительные требования

- Node.js 18+
- Python 3.12+
- Playwright

### Фронтенд

```bash
npm install
npm run dev
```

Открыть http://localhost:5173

### Бэкенд

```bash
cd backend
pip install -r requirements.txt
playwright install chromium
python manage.py init
python manage.py seed
python run.py
```
