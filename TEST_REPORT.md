## ШАГ 2: ПРОВЕРКА СУЩЕСТВУЮЩИХ ТЕСТОВ ✅

### Исправленные проблемы

1. **Async fixture bug** - исправлено
   ```python
   # Было:
   @pytest.fixture
   async def page(context):
       page = await context.new_page()
       yield page  # ❌ Ошибка
       await page.close()
   
   # Стало:
   @pytest.fixture
   async def page(context):
       page = await context.new_page()
       try:
           yield page  # ✅ Правильно
       finally:
           await page.close()
   ```

2. **Scope mismatch** - исправлено
   - Убран `scope="session"` у browser fixture
   - Добавлен `pytest.ini` с правильными настройками

3. **pytest-asyncio настройка**
   ```ini
   [pytest]
   asyncio_mode = auto
   asyncio_default_fixture_loop_scope = function
   ```
