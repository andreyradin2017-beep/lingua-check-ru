# Product Spec: LinguaCheck‑RU

## 1. Назначение системы

Система автоматически проверяет:

- сайты (по URL) и
- тексты (ввод/загрузка файлов)  
  на соответствие требованиям к использованию русского языка в публичном пространстве (ФЗ №168‑ФЗ):
- приоритет русского языка;
- отсутствие иностранных слов, не включённых в нормативные словари;
- визуальное доминирование русского текста.

Система внутренняя (для личных целей, без коммерции).

---

## 2. Ключевые сущности

- `Project` — проверка одного сайта или текста.
- `Scan` — запущенный процесс проверки сайта.
- `Page` — страница сайта.
- `Token` — токенифицированное слово.
- `Violation` — найденное нарушение.
- `Trademarks` — список слов‑товарных знаков (не считаются нарушением, но помечаются).

---

## 3. Режимы работы

### 3.1. Проверка сайта по URL

1. Пользователь вводит URL.
2. Система парсит сайт (с JS‑рендерингом, по `sitemap.xml` и `robots.txt` по опции).
3. Система извлекает видимый текст и CSS‑стили для каждого текстового элемента.
4. **Языковая проверка (Phase 9)**: Система оценивает плотность кириллицы на странице. Если страница признана нерусскоязычной (кириллицы < 10%), ее анализ пропускается.
5. Система анализирует слова (по словарям + морфологии) и визуальное доминирование.
6. Система формирует отчёт со скриншотами нарушений (Phase 6).
7. **Фильтрация и исключения (Phase 8)**: Игнорирование имен собственных, городов и сокращений. Латинские слова с заглавной буквы помечаются как `possible_trademark`.

### 3.2. Проверка текста

1. Пользователь вводит текст или загружает файл (TXT/DOCX/PDF).
2. Система при необходимости применяет OCR (для PDF), токенизирует текст.
3. Система анализирует слова по нормативным словарям, морфологически через `pymorphy2`.
4. Система формирует отчёт по тексту.

---

## 4. Нормативная база и словари

- Федеральный закон №168‑ФЗ «О защите русского языка как государственного языка РФ».
- Перечень нормативных словарей (Распоряжение Правительства №1102‑р):
  - Орфографический словарь русского языка как государственного языка РФ;
  - Орфоэпический словарь;
  - Словарь иностранных слов;
  - Толковый словарь государственного языка РФ;
  - и другие из официального перечня.

Словари берутся из PDF‑версий на https://ruslang.ru/normativnyje_slovari:

- PDF → OCR → CSV → таблица `dictionary_words`.

---

## 5. Требования к функциям

- Текст нормализуется (пробелы, знаки препинания).
- Все слова лемматизируются с помощью `pymorphy2`; в словаре ищется `normal_form`.
- Если лемма есть в `dictionary_words` — слово **допустимо**.
- Если нет и нет русского дубля в том же блоке — **потенциальное нарушение**.
- Визуальный вес элемента считается по:
  - `font_size` (px),
  - `font_weight` (400/700),
  - `contrast_ratio` (контраст цвет/фон, WCAG‑подход).

Формула визуального веса:

visual*weight = 0.6 * font*size_px + 0.3 * font_weight + 0.1 \* contrast_ratio

Правило доминирования:

- если в одном блоке пара foreign vs ru и
  visual_weight_foreign > 1.5 \* visual_weight_rus
  → нарушение типа `visual_dominance`.

---

## 6. Ограничения

- Словари обновляются вручную (новые PDF не подхватываются автоматически).
- Система не даёт юридически обязательных заключений.
- Использование — внутреннее, без коммерческой продажи сервиса.
