# Test Data Spec

## 1. Тест‑сайт

Создать маленький тестовый сайт (статические HTML‑страницы):

- `index.html`:
  - крупный текст: "COFFEE TO GO" (font-size: 40px, bold),
  - мелкий подпись: "Кофе с собой" (font-size: 12px).
- `promo.html`:
  - рекламный слоган на английском без русского дубля.

Цель:

- Проверить визуальное доминирование и отсутствие русского дубля.

## 2. Тест‑тексты

### test_text_1.txt

- Примеры:
  - «Я купил новую батарею для своего устройства».
  - «Теперь я использую супер‑гаджет CoffeeMaster Pro».

Ожидание:

- `батарею` → лемма `батарея` — находится в словарях.
- `CoffeeMaster` — отсутствует в словарях, помечается как потенциально иностранное (если не в trademarks).

### test_text_2.txt

- Набор фраз только с нормативными словами, чтобы убедиться, что нарушений нет.

## 3. Тест‑кейсы

- Лемматизация:
  - подтверждение, что формы `батарея / батарею / батареи` маппятся на одну лемму.
- Визуальный анализ:
  - `COFFEE TO GO` vs `Кофе с собой` → `visual_dominance`.
- Сканирование:
  - обработка таймаутов и заблокированных страниц без падения всего скана.
