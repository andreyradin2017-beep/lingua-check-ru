**Выполнено:**
- ✅ Retry logic с экспоненциальной задержкой
- ✅ 3 попытки (1s, 2s, 4s, max 10s)
- ✅ UI компонент для отображения ошибок
- ✅ Кнопка "Повторить" для retry
- ✅ Разные сообщения для 404, 500 ошибок

**Пример:**
```tsx
const checkStatus = useCallback(async (id: string, retryCount = 0) => {
  try {
    // ... запрос к API
  } catch (err) {
    // Retry с экспоненциальной задержкой
    if (retryCount < 3) {
      const retryDelay = Math.min(1000 * Math.pow(2, retryCount), 10000);
      setTimeout(() => checkStatus(id, retryCount + 1), retryDelay);
    }
  }
}, []);
```
