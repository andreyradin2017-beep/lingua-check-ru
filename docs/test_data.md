## 3. Тест-кейсы

### TC-001: Лемматизация

**Цель:** Проверить корректность лемматизации.

**Входные данные:**
```
батарея, батарею, батареи, батарее, батареей, батарее
```

**Ожидаемый результат:**
Все формы маппятся на лемму `батарея`.

---

### TC-002: Исключения (Trademarks)

**Цель:** Проверить работу исключений.

**Шаги:**
1. Добавить бренд: `CoffeeMaster`
2. Запустить проверку текста с `CoffeeMaster`
3. Проверить отчет

**Ожидаемый результат:**
- Слово помечается как `trademark`
- Не считается нарушением

---

### TC-003: Глобальные исключения

**Цель:** Проверить работу global_exceptions.

**Шаги:**
1. Добавить исключение: `gmp`
2. Запустить сканирование с словом `gmp`
3. Проверить отчет

**Ожидаемый результат:**
- Слово не помечается как нарушение
- Исключение применяется глобально

---

### TC-004: Safe Tokenizer (версия 1.13.0+)

**Цель:** Проверить исключение технических терминов.

**Входные данные:**
```
drug - технический термин
getElementById - JavaScript метод
javascript - язык программирования
```

**Ожидаемый результат:**
- `drug` не помечается как нарушение
- `getElementById` не помечается как нарушение
- `javascript` не помечается как нарушение

---

### TC-005: Группировка нарушений (версия 1.9.0+)

**Цель:** Проверить группировку нарушений.

**Шаги:**
1. Запустить сканирование сайта с повторяющимися нарушениями
2. Получить сгруппированные данные через `/api/v1/scan/{id}/grouped`
3. Проверить формат ответа

**Ожидаемый результат:**
```json
[
  {
    "word": "нутриентах",
    "count": 80,
    "type": "unrecognized_word",
    "page_urls": ["https://example.com/page1"]
  }
]
```

---

### TC-006: Экспорт отчетов (версия 1.13.0+)

**Цель:** Проверить экспорт в различные форматы.

**Шаги:**
1. Запустить сканирование
2. Дождаться завершения
3. Экспортировать в Excel
4. Экспортировать в PDF
5. Проверить имена файлов

**Ожидаемый результат:**
- XLSX: `linguacheck_{timestamp}.xlsx`
- PDF: `linguacheck_{timestamp}.pdf`
- Корректные заголовки `Content-Disposition`

---

### TC-007: Smart Crawler (версия 1.13.0+)

**Цель:** Проверить фильтрацию скрытых блоков.

**Входные данные:**
```html
<div aria-hidden="true">
  HIDDEN CONTENT with foreign words
</div>
```

**Ожидаемый результат:**
- Содержимое `aria-hidden="true"` игнорируется
- Нарушения не фиксируются

---

### TC-008: In-Memory Caching (версия 1.9.0+)

**Цель:** Проверить кэширование словарей.

**Шаги:**
1. Запустить первое сканирование
2. Запустить второе сканирование
3. Сравнить время выполнения

**Ожидаемый результат:**
- Первое сканирование: загрузка из БД
- Второе сканирование: использование кэша
- Ускорение: в 10 раз
